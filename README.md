# AI Resume Analyzer - Backend

FastAPI backend for analyzing resumes using AI (Gemini API).

---

##  Features

-  Resume PDF parsing
-  AI-powered analysis (Gemini)
-  ATS score generation
-  FastAPI performance
-  Basic API protection
-  Rate limiting (SlowAPI)

---

## Tech Stack

- FastAPI
- Uvicorn
- Google Gemini API
- pdfplumber
- BeautifulSoup
- SlowAPI
